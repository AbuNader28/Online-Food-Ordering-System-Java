# Online Food Ordering System 🍔🍕

## 📌 Project Overview
A fully functional Online Food Ordering System developed in **Java** as a final academic project for the Advanced Programming course. This project demonstrates core computer science principles and software development practices.

## 🚀 Features
* **User-Friendly Ordering:** Browse menus, select items, and place orders smoothly.
* **Order Management:** Efficient processing of customer requests and dynamic cart updates.
* **Robust Architecture:** Built with clean code practices to ensure scalability and maintainability.

## 💻 Technologies & Concepts Used
* **Language:** Java
* **Core Concepts:** Object-Oriented Programming (OOP)
* **Data Handling:** Implemented fundamental **Data Structures** for efficient menu and order tracking.
